# Accounting Rules Label Update

Updated the visible naming from **Rule Templates (Advanced)** / **Accounting Rule Templates** to **Accounting Rules**.

## Changed

- Sidebar menu label now shows `Accounting Rules`.
- Accounting rule page title now shows `Accounting Rules`.
- Add/Edit modal labels now show `Add Accounting Rule` and `Edit Accounting Rule`.
- Save button now shows `Save Accounting Rule`.
- Empty state now shows `No accounting rules found.`
- Related backend validation/setup messages now use `accounting rule` wording instead of `rule template`.

## Not Changed

- Routes remain the same: `accounting-rules.*`.
- Database tables/models remain the same.
- Accounting posting logic remains unchanged.
