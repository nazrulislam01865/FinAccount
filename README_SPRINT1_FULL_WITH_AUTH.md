# Sprint 1 Full Package With Auth

This package includes the Sprint 1 frontend, backend, dynamic dropdown APIs, migrations, seeders, and Breeze-style authentication files.

Copy into a fresh Laravel project after installing dependencies, or copy the full contents over an existing Laravel project. This package includes:

- routes/web.php
- routes/api.php
- routes/auth.php
- routes/console.php
- app/Http/Controllers/Auth/*
- app/Http/Requests/Auth/LoginRequest.php
- resources/views/auth/*
- Sprint 1 setup modules

- Email: admin@example.com
- Password: password
